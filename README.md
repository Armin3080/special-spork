# 🚀 X DOWNLOADER - حرفه‌ای‌ترین دانلودر

یک دانلودر قدرتمند و حرفه‌ای با پایتون، قابلیت دانلود از هر سایتی با استفاده از yt-dlp، ffmpeg و requests.

## ✨ قابلیت‌ها

- ⚡ دانلود با سرعت بالا
- 🎬 پشتیبانی از تمام سایت‌های معروف (YouTube, Instagram, Aparat, SoundCloud, و...)
- 🎵 استخراج صدا از ویدیوها
- 🔍 استخراج خودکار لینک از صفحات وب
- 📊 نمایش سرعت و زمان باقیمانده
- 🎨 گرافیک مدرن و شیشه‌ای
- 🌍 دو زبانه (فارسی/انگلیسی)
- 💾 محدودیت حجم ۲ گیگابایت
- 🔄 قابلیت Resume دانلود
- 📱 کاملاً ریسپانسیو

## 📋 پیش‌نیازها

- Python 3.8 یا بالاتر
- pip
- ffmpeg (در پوشه پروژه قرار دارد)
